# autoProover

## Алгоритм
В алгоритме используется построение дерева секвенций - начальная построенная секвенция доказывается посредством доказательства листов полученного из этой секвенции дерева.
Для доказательства выводимости секвенций используются правила:
- modus_ponens;
- теорема о дедукции;
- дополнительное правило избавления от отрицаний.
